# Development of an R-based Analytical Pipeline for Non-Targeted Detection and Chemical Profiling of Illicit Drug Waste in Environmental Samples

## Overview

This repository contains the R workflow developed for the analysis of non-targeted LC-MS data from environmental samples associated with illicit drug waste.

The workflow includes:

- Data import from MS-DIAL output files
- Log2 transformation
- Median normalization
- Multidimensional scaling (MDS) analysis
- Suspect screening using MetaboAnnotation
- Heatmap visualization

## Repository Structure

scripts/ - Analysis scripts

metadata/ - Suspect database and metadata

results/ - Figures generated during analysis

manuscript/ - Final manuscript

## Software Requirements

- R (version 4.3 or higher)
- MetaboAnnotation
- MetaboCoreUtils
- limma
- tidyverse
- edgeR
- Glimma
- viridis

## Data Availability

The raw LC-MS data used in this study are not publicly available due to confidentiality restrictions related to environmental forensic investigations.
